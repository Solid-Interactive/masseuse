Class: ComputedProperty

ComputedProperty

new ComputedProperty(listenablePropertyNames, callback, skipInitialComputation)

A ComputedProperty is a property that depends on one or more other model values. Each time one of the other values is changed, the ComputeProperty callback is called with the values of all the dependent fields. See the tests for example usage.
