<static> start($parentRenderPromise) → {jQuery.promise}

Wrapper method for lifecycle methods. Life cycle event are notifed both throw the progress returned by this method's promise, and by events triggered on the view. So - for example - plugins can hook into life cycle events. In order, this method: - Calls this.beforeRender() - Starts any child views - Notifies that beforeRender is done - If the view has a parent, waits for its parent to render - Calls this.render() - Notifies that render is done - Calls this.afterRender() - Notifies that afterRender is done - Resolves the returned promise
Parameters:
Name Type Description
$parentRenderPromise jQuery.promise If passed in, the start method was called from a parent view start() method.

A promise that is resolved when when the start method has completed
